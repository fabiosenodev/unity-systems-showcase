# Unity Systems Showcase (C#)

Modular Unity C# systems for real-time 3D applications: input, movement, interaction, camera control, and stamina management.

## What this repo contains
- **Scripts/Input** – input handling patterns (Unity Input System-ready)
- **Scripts/Movement** – locomotion / controller logic
- **Scripts/Interaction** – interact detection + prompt logic
- **Scripts/Camera** – third-person camera control (smooth follow, rotation, clamping)
- **Scripts/Utilities** – helpers used across systems

## Why this repo exists
Built to demonstrate clean, reusable Unity gameplay/engine systems with readable code structure and performance-conscious patterns.
